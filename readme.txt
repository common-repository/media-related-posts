== Description ==

This plugin works in admin. It shows a new column in media.
This new column shows related posts (in custom fields, post content and post thumbnail) for each media.

Available languages : english (default), french

== Installation ==

1. Upload 'media-related-posts' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to media to see the new column 'Related Posts'
